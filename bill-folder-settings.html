入れるファイルの種類

報告

作業手順

  1. ローカル作業フォルダー内に gian フォルダーを作成する
  2. その中に回次フォルダー（例: 446）を作成する
  3. 議案・報告・提案理由・施政方針のPDFを、既定ファイル名のまま入れる
  4. 必要に応じて GitHub へコミット・プッシュして GitHub Pages へ反映する
