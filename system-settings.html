システム設定

議会情報の表示内容は「今議会設定（Webスプレッドシート）」から更新できます。

今議会設定

下のボタンを押すと、ブラウザ上で議会情報を編集できます。保存すると今議会の資料ページの赤枠情報へ反映されます。

今議会設定を開く

ログインユーザー管理

ログイン可能ユーザーの権限（viewer/editor/admin）を登録・変更します。

ユーザー管理を開く

議案PDFのローカル登録

議案・報告・提案理由・施政方針のPDFを、ローカル作業フォルダー内の所定フォルダーへ入れるための入口です。保存先フォルダー名と回次ごとの配置先を確認できます。

議案PDF登録ページを開く

議員・職員登録

議員および事務局スタッフの基本情報を登録・更新します。役職はマスター管理で追加・削除できます。

議員・職員登録画面を開く

テスト公開と本番移行

GitHub Pages でのテスト公開、Supabase 接続、Google ログイン設定、本番移行手順を文書化しています。

ワークスペース内の DEPLOY_GUIDE.md を参照してください。
